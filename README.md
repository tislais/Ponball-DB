# Ponball-DB

Scrapes IPDB for these values:

```
✓ title,
✓ ipdb id,
✓ manufacturer,
✓ manufacture date,
✓ mpu,
✓ type,
✓ production,
✓ theme,
✓ specialty,
✓ features,
✓ design by,
✓ art by,
✓ dots/animation by,
✓ mechanics by,
- music_by,
✓ sound by,
✓ software by,
✓ notes,
✓ rule sheets
✓ roms
✓ documentation,
- images
```
